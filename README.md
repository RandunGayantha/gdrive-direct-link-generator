# 🚀 GDrive Direct Link Generator

**Live Link:** [https://randungayantha.github.io/gdrive-direct-link-generator/](https://randungayantha.github.io/gdrive-direct-link-generator/)

---

A lightweight, fast, and dark-themed web tool to convert standard Google Drive "Share" links into **Direct Download** links. 

Perfect for bloggers, subtitle creators, and developers who want to provide a "one-click" download experience for their users.

## ✨ Features
- **Instant Conversion:** Converts links locally using JavaScript.
- **One-Click Copy:** Integrated "Copy to Clipboard" button.
- **Direct Download:** Bypasses the preview screen to force a download.

## 🛠️ How to Use
1. **Get your link:** Right-click your file in Google Drive, select **Share**, and set permissions to "Anyone with the link."
2. **Convert:** Paste the link into this tool.
3. **Download:** Use the generated link in your Blogger site.
